### 🌟 Overview
**ProTodo** is a high-end, production-ready task orchestration dashboard engineered for modern web browsers. Designed with a strict **Nature & Premium UI/UX architectural theme**, it balances deep visual elegance with performance. Built upon vanilla web engineering standards, it seamlessly handles atomic CRUD workflows and incorporates robust backend sessions.

> 🌐 **Live URL:** [https://pro-todo-app.vercel.app/](https://pro-todo-app.vercel.app/)

---

### ⚡ Key Engineering Features

*   **🛡️ Multi-Provider OAuth 2.0 Integration:** Full corporate handshake pipelines using Google & GitHub authentication protocols managed via Supabase.
*   **🔒 Session Routing Guards:** Structural client-side token verification patterns that catch unauthorized manual URL manipulation (`todo.html`) and cycle users back gracefully.
*   **🎨 Premium Deep Cyan Aesthetics:** High-end dark ambient layouts, premium glassmorphism accents, fluid loader animations, and SweetAlert2 dynamic feedback modals.
*   **🔄 Fully Asynchronous Real-time CRUD:** Isolated state updates mapping active task insertions, updates, and hard deletions back to relational cloud instances instantly.

🛠️ Tech Stack

- Frontend: HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
- Backend: Supabase (Auth + Database)
- Authentication: OAuth 2.0 (Google & GitHub)
- Deployment: Vercel
- UI Alerts: SweetAlert2

🔐 Security Features

- Row Level Security (RLS)
- Auth-based data filtering ("auth.uid()")
- Protected routes (session validation)
- OAuth secure login flows
